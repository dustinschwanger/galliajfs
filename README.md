# Gallia County Department of Job & Family Services Website

Official website for the Gallia County Department of Job and Family Services providing information about public assistance, child support, child and adult protective services, and workforce development.

## Features

- **Responsive Design**: Mobile-first design that works on all devices
- **WCAG 2.2 AA Compliant**: Fully accessible to users with disabilities
- **USWDS Framework**: Built on the U.S. Web Design System for consistency and accessibility
- **Interactive Chat Widget**: User-friendly chat interface for quick assistance
- **Organized Service Information**: Easy-to-navigate accordion sections for key services

## Services Covered

- Public Assistance (SNAP, Medicaid, Cash Assistance)
- Child Support Enforcement
- Child & Adult Protective Services
- Workforce Development
- Veteran Services
- Child Care Assistance
- Foster Care & Adoption

## Technology Stack

- HTML5
- CSS3 (Custom styles + USWDS)
- Vanilla JavaScript (no framework dependencies)
- Font Awesome Icons
- USWDS 3.7.1

## Deployment

This site is configured for deployment on Railway with Python HTTP server.

### Deploy to Railway

1. Connect this repository to Railway
2. Railway will automatically detect the configuration
3. The site will be served on port 8080

### Local Development

To run locally:

```bash
python -m http.server 8080
```

Then visit http://localhost:8080
